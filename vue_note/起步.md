# 起步
---
### 引包
    将vue.js引入文件中,使文件具备vue对象。
### 启动vue
```javascriot
new Vue(options);
```
### 配置options对象
```javascript
new Vue ({
    el: '#app', //发生行为的目的地
    // el: document.getElementById('app');
    template: '<div><h1>好好学习，{{text}}</h1></div>',//装载/插入的模板
    data: function() {//动态数据的声明
        return {
            text: '天天向上！'
        }
    }
})
```
### 插值表达式 {{表达式}}
> 表达式应用的变量必须在data中声明
- 对象：{{ {name: 'Ethan'} }}
- 字符串：{{'my name is Ethan.'}}
- 判断后的布尔值：{{true}}
- 三元表达式：{{true?a:b}}
